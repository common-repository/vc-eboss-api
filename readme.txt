EBoss Recruitment Software WEB API Integration for Wordpress

== Description ==

EBoss Web Api work with EBoss Recruitment Software. The Credentials are provided by the EBoss Recruitment Software Company and these can be integrated to your existing website through API. This plugin is created to integrate basic functionalities available with the API to your wordpress based website.

[EBoss Recruitment Software](https://www.ebossrecruitment.com/)

== Installation ==

The plugin is created on [Herbet Framework](http://www.getherbert.com/) which is based on [Laravel Framework](https://laravel.com/).

1. Upload plugin files to your plugins folder.
1. Use [composer](https://getcomposer.org/) to extract required plugin files.
1. Connect your server through SSH.
1. There after route through path wp-content/plugin/vc-eboss-api.
1. Then run composer update.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Select "eBoss+ Recruitment API" from the left-hand admin menu and add "eBoss+" API credentials

== Frequently Asked Questions ==

= What is eBoss+ Recruitment Software? =

eBoss+ Recruitment Software is a personalized recruitment service used by majority companies to handle recruitment at their companies.

= What does eBoss API Recruitment do ? =

It integrates your eBoss Recruitment Software with your website using it's API services.

= What do I need to use eBoss Recruitment Software Eboss API Plugin? =

You need to install composer to install require plugin files, also required credentials from eBoss+ to activate.
